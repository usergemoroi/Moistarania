# Changelog

Все заметные изменения в этом проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),

## [1.0.0] - 2024-02-01

### Добавлено
- ✨ Первоначальный релиз Android инжектора для Standoff 2
- ✨ External ESP overlay с поддержкой OpenGL ES 3.0
- ✨ Простой Material Design 3 интерфейс
- ✨ Автоматическая проверка и запрос root прав
- ✨ Сборка native библиотеки на устройстве
- ✨ Поддержка prebuilt binary для устройств без NDK
- ✨ Автоматический запуск Standoff 2 из приложения
- ✨ Service для управления overlay процессом
- ✨ Базовый ESP с отображением:
  - Box ESP (рамки вокруг игроков)
  - Health Bar (полоса здоровья)
  - Team ESP (различие команд)
  - World-to-Screen проекция
- ✨ ImGui меню для настройки overlay
- ✨ Поддержка Android 5.0+ (API 21+)
- ✨ Поддержка ARM64-v8a архитектуры
- ✨ Детальная документация:
  - README.md с общим описанием
  - INSTALL.md с инструкциями по установке
  - USAGE.md с руководством по использованию
  - BUILD.md с инструкциями по сборке
- ✨ Скрипт build_native.sh для предварительной сборки
- ✨ Gradle конфигурация для автоматизированной сборки
- ✨ .gitignore для правильной работы с Git

### Технические детали
- **Android App:**
  - Язык: Kotlin
  - Min SDK: 21 (Android 5.0)
  - Target SDK: 34 (Android 14)
  - Архитектура: MVVM с Service

- **Native Overlay:**
  - Язык: C++17
  - Рендеринг: EGL + OpenGL ES 3.0
  - GUI: Dear ImGui v1.89
  - Сборка: Android NDK (ndk-build)

- **Возможности:**
  - process_vm_readv/writev для чтения памяти
  - ANativeWindow для overlay окна
  - Linux input/uinput для обработки касаний
  - IL2CPP структуры для чтения игровых данных

### Известные проблемы
- ⚠️ Требуется root доступ (не работает без root)
- ⚠️ Первая сборка может занимать 5-10 минут на устройстве
- ⚠️ Не тестировалось на всех устройствах
- ⚠️ Offsets могут устареть после обновления игры
